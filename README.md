## Skate React

## Introduction: 
A fun and interactive single page skateboard shop web application, using the MERN stack, that has create, read, and delete user functionality. 

## Technologies Used 
* HTML 5
* CSS 3
* SASS
* JavaScript
* MongoDB
* Express.js
* React
* Node.js
* Google Fonts
* Google firebase
